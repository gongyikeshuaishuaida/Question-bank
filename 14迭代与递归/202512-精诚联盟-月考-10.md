# 题目 10

如下Python 程序段，实现十进制数转十六进制数:

```python
def cal(dec):
    chars = "0123456789ABCDEF"

    if dec == 0:
        return ""
    else:
        return          ▲

n = int(input("请输入待转换的十进制数:"))
print(cal(n))
```

方框中应填入的正确代码为

A. `cal(dec//16)+chars[dec%16]`
B. `chars[dec//16]+cal(dec%16)`
C. `cal(dec%16)+chars[dec//16]`
D. `chars[dec%16]+cal(dec//16)`

---

## 答案

**正确答案：** A

---

## 解析

十进制转十六进制时，需要先递归处理高位 `dec//16`，再拼接当前最低位 `chars[dec%16]`。例如 `31` 应输出 `1F`，符合 `cal(dec//16)+chars[dec%16]`。
