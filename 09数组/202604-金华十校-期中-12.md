# 题目 12

有如下程序段：

```python
# 输入列表 a 和 k，代码略
n = len(a)
j = 0
f = [0] * n
for i in range(1, n):
    while j < i and a[i] - a[j + 1] > k:
        j += 1
    f[i] = f[j] + 1
print(f[n - 1])
```

若输入的 `a=[0,1,3,5,7,8,9,10]`，`k=2`，执行该程序段后，输出的是（   ）

- A. 4
- B. 3
- C. 2
- D. 1

---

## 答案

**正确答案：** B

---

## 解析

逐步跟踪 `i` 与 `j`：`f[1]=1`，`f[2]=1`；到 `i=3` 时 `j` 增至 1，`f[3]=f[1]+1=2`；`i=4` 时 `j` 增至 2，`f[4]=f[2]+1=2`；`i=5` 时 `j` 增至 3，`f[5]=f[3]+1=3`；`i=6` 时 `j` 不变，`f[6]=3`；`i=7` 时 `j` 增至 4，`f[7]=f[4]+1=3`。最终输出 `f[n-1]`，即 3。
