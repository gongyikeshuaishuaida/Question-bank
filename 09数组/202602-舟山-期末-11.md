# 题目 11

有如下 Python 程序段：

```python
signal = [5, 3, 6, 2, 4, 1, 2, 2]
kernel = [1, 2, 3]
out = []
for i in range(len(signal) - len(kernel) + 1):
    t = 0
    for j in range(len(kernel)):
        t += signal[i + j] * kernel[j]
    out.append(t)  # 向列表out中添加元素t
```

执行该程序段后，`out[2]` 的值为（   ）

- A. 29
- B. 21
- C. 13
- D. 22

---

## 答案

**正确答案：** D

---

## 解析

`out[2]` 对应 `i=2` 时的计算：`signal[2]*1 + signal[3]*2 + signal[4]*3 = 6*1 + 2*2 + 4*3 = 22`。
