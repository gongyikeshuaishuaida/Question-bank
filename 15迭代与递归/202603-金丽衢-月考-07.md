# 题目 07

某 Python 程序如下：

```python
def f(a,b):
    if b == 0:
        return 1
    elif b % 2 == 0:
        return f(a*a, b//2)
    else:
        return f(a*a, b//2) * a

result = f(2,10)
print(result)
```

执行该程序后，输出值为（   ）

- A. 1024
- B. 32
- C. 64
- D. 512

---

## 答案

**正确答案：** A

---

## 解析

函数用递归方式计算 `a^b`。`f(2,10)` 等价于 `2^10`，结果为 1024。也可跟踪为 `f(2,10)->f(4,5)->f(16,2)*4->f(256,1)*4->f(65536,0)*256*4=1024`。
