# 题目 10

有如下 Python 程序段：

```python
d = [[2,4],[7,3],[4,-1],[1,2],[8,1]]
head = 0
p = head
while p != -1 and d[p][1] != -1:
    q = d[p][1]
    if d[p][0] % 2 == d[q][0] % 2:
        d[p][1] = d[q][1]
    else:
        p = d[p][1]
```

执行该程序段后，从 `head` 开始按链表顺序访问 `d`，得到的数据序列是（   ）

- A. `2,8,7,1,4`
- B. `2,7,4`
- C. `2,8,4`
- D. `8,1,4`

---

## 答案

**正确答案：** B

---

## 解析

初始链表顺序为 `2 -> 8 -> 7 -> 1 -> 4`。程序删除与当前节点奇偶性相同的后继节点：`2` 与 `8` 同为偶数，删除 `8`；随后 `7` 与 `1` 同为奇数，删除 `1`。最终链表为 `2 -> 7 -> 4`。
