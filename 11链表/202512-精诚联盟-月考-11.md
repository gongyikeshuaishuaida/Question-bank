# 题目 11

有如下Python 程序段：

```python
from random import randint
k = randint(2, 5)
lst = [["A",3],["C",4],["D",5],["E",6],["B",7],["G",0],["F",2],["N",5]]
head = 1; p = q = head; i = 0
while i <= k:
    p = lst[p][1]; q = lst[lst[q][1]][1]
    if p == q:
        break
    i += 1
```

运行后，lst[q][0]的值不可能是

A. `"G"`
B. `"A"`
C. `"E"`
D. `"F"`

---

## 答案

**正确答案：** B

---

## 解析

该链表存在环，`p` 每轮走一步，`q` 每轮走两步。按循环跟踪，`k` 可能为 2、3、4、5，且遇到 `p == q` 会提前退出，最终 `lst[q][0]` 可能为 `"F"`、`"G"` 或 `"E"`，不可能为 `"A"`。
