# 题目 11

有如下 Python 程序段：

```python
tag = [0] * len(data)
i = 0
p = i
while i < len(data):
    if p != -1:
        tag[i] += 1
        p = data[p][1]
    else:
        i += 1
        p = i
```

若 `data` 为 `[[0,1],[1,2],[2,-1],[3,2],[4,3]]`，运行该程序段后，`tag[4]` 的值为（   ）

A. 1  
B. 2  
C. 3  
D. 4

---

## 答案

**正确答案：** C

---

## 解析

当 `i=4` 时，`p` 依次为 4、3、2、-1，循环中在 `p!=-1` 的三次迭代里都执行 `tag[4]+=1`，因此 `tag[4]` 的值为 3。
