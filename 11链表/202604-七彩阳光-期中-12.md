# 题目 12

在 Python 中，用列表模拟一个存储机器人动作指令的链表。每个节点 `[data,next]` 包含两个元素，其中 `data` 存储指令名称，`next` 存储下一个节点的索引。现有如下代码：

```python
nodes = [["前进", 4], ["左转", 0], ["左转", 3], ["右转", -1], ["后退", 2]]
head = 1
p = head
prev = -1
while p != -1:
    if nodes[p][0] == "左转":
        if prev == -1:
            head = nodes[p][1]
        else:
            nodes[prev][1] = nodes[p][1]
    else:
        prev = p
    p = nodes[p][1]

p = head
count = 0
while p != -1:
    count += 1
    if count == 3:
        print(nodes[p][0])
        break
    p = nodes[p][1]
```

执行该程序后，输出的结果是（   ）

- A. `"前进"`
- B. `"左转"`
- C. `"后退"`
- D. `"右转"`

---

## 答案

**正确答案：** D

---

## 解析

原链表从 `head=1` 开始，顺序为 `左转 -> 前进 -> 后退 -> 左转 -> 右转`。第一段程序会删除所有 `"左转"` 节点，删除后链表为 `前进 -> 后退 -> 右转`。第二段程序输出删除后链表的第 3 个节点数据，因此输出 `"右转"`。
