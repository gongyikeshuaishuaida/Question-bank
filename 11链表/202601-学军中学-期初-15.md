# 题目 15

某学校信息技术老师根据每位学生历次考试的第 13、14、15 题平均得分，为每道题平均分排在年级后 50% 的学生挑选针对性的试题进行练习。假设每道题的学生平均得分无并列，如果学生数是奇数则选出一半多一个。

针对性的试题分别由两位老师从题库中挑选，学生练习的试题是这两位老师挑选的试题的交集（即被两位老师同时选中的题目，并假设该交集非空）。部分学生历次考试各题均分如图所示。

![[attachments/202601学军中学_15_图1.png]]

用 Python 编写程序模拟实现上述问题，输出结果如图所示，请回答下列问题：

（1）如果其中 1 位老师从题库中选取第 15 题的试题编号为 `N6,N8,N11,N15,N19,N25`，另外一位老师从题库中选取第 15 题的试题编号为 `N3,N5,N6,N15,N17,N19,N30`，则第 15 题给学生训练的试题数量是____个。

（2）实现为每道题平均分排在年级后 50% 的学生挑选针对性试题进行练习的程序如下，请在划线处填入合适的代码。

（3）`findcommon(t)` 函数实现找出两位老师针对该题目所选出的相同的题号。加框处代码有误，应该修改为____。

```python
def findcommon(t):
    p, q = headA[t], headB[t]
    ans = []
    while p != -1 and q != -1:
        sa = trainA[p][0]; sb = trainB[q][0]
        if sa == sb:
            ans.append(sa)
            p = trainA[p][1]
            q = trainB[q][1]
        elif sa < sb:
            p = trainA[p][1]
        else:
            q = trainB[q][1]
    return ans
```

---

## 答案

（1）3

（2）① `p != -1 and stu[p][i + 2] < stu[j][i + 2]`；② `p != w[slow][i]`；③ `ans = findcommon(i)`

（3）`int(sa[1:]) < int(sb[1:])`

---

## 解析

第 15 题两个集合交集为 `N6、N15、N19`，数量为 3。构造链表时按第 13、14、15 题均分升序插入，输出前半部分学生；题号形如 `N6`、`N15`，直接字符串比较会出现字典序错误，应比较去掉 `N` 后的数值。
