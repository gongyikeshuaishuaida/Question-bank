# 题目 07

某算法的部分流程图如下图所示，若 n 的值为 6，数组元素 `a[0]` 至 `a[n-1]` 依次存放 `2, 8, 4, 8, 6, 2`，执行这部分流程后，输出 k 的值为（   ）

![[attachments/202604湖衢丽_07_图1.png]]

- A. 0
- B. 1
- C. 3
- D. 5

---

## 答案

**正确答案：** C

---

## 解析

流程从 `i=5,k=5` 开始，每次先令 `i=i-1`，若 `a[k]<a[i]` 则更新 `k=i`。依次比较得到：`a[5]=2<a[4]=6`，`k=4`；`a[4]=6<a[3]=8`，`k=3`；之后遇到 `4、8、2` 均不满足严格小于当前最大值 8，所以最终输出 `k=3`。
