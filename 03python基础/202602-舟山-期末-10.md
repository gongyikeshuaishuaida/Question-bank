# 题目 10

有如下 Python 程序段：

```python
s = "110,1001,101,1101,"
t = m = 0
for i in s:
    if i in "01":
        t = t * 2 + int(i)
    else:
        if t > m:
            m = t
        t = 0
```

执行该程序段后，`m` 的值为（   ）

- A. 13
- B. 9
- C. 110
- D. 1101

---

## 答案

**正确答案：** A

---

## 解析

程序逐段把二进制字符串转为十进制数，并记录最大值。`110`、`1001`、`101`、`1101` 分别为 6、9、5、13，所以 `m` 最终为 13。
