# 题目 10

有如下 Python 程序段：

```python
chs = input("请输入字符串：")
s = []
res = ""
for ch in chs:
    if s != [] and s[-1] == ch:
        s.pop()
    else:
        s.append(ch)
for i in range(len(s)):
    res += s[i]
print(res)
```

程序运行时，若输出为 `care`，则可能的输入是（   ）

A. `cbbare33e`  
B. `cbeare33b`  
C. `caeer33rb`  
D. `caar33reb`

---

## 答案

**正确答案：** A

---

## 解析

程序用列表 `s` 保存当前结果，遇到与栈顶相同的字符时弹出，否则入栈。输入 `cbbare33e` 时，两个 `b` 相邻抵消，两个 `3` 相邻抵消，剩余字符为 `care`，因此选 A。
