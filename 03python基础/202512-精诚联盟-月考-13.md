# 题目 13

小明整理了数篇高考英语阅读理解的文章，如第13 题图所示，统计这些文章中词汇首字母出现的频率，最终输出出现频率最高首字母，若存在多个频率相同字母一并输出。

![[attachments/202512精诚联盟_13_图1.png]]

（1）若小写字母a 到f 为首字母的出现频率依次为88、85、75、70、88、87，则最终选出的首字母为（填字母，逗号分隔）：     ▲

（2）实现上述功能的部分Python 程序如下，请在划线处填入合适的代码。

```python
# 读取所有文章内容，均转为小写字母后，存储于变量line 中
d = {}; i = 0; word = ""
while i <= len(line)-1:
    ch = line[i]
    if "a" <= ch <= "z":
        word += ch
    else:
        if len(word) > 0:
            if word not in d:
                ①
            d[word] += 1
        word = ""
    i += 1
s = [0 for i in range(26)]
for i in d:
    ②
m = 0; sign = []
for i in range(26):
    if s[i] > m:
        m = s[i]; sign = [chr(i+97)]
    ③:
        sign.append(chr(i+97))
print("出现频率最高首字母是", sign)
```

---

## 答案

**正确答案：** （1）a，e；（2）① `d[word]=0`；② `s[ord(i[0])-97]+=d[i]`；③ `elif s[i]==m`

---

## 解析

（1）a 和 e 的出现频率均为 88，是给定频率中的最大值，所以都要输出。（2）遇到新单词时先在字典中初始化计数为 0，再统一执行 `d[word] += 1`；统计首字母频率时，单词 `i` 的首字母索引为 `ord(i[0])-97`，累加该单词出现次数 `d[i]`；遍历 26 个字母时，若频率等于当前最大值，应使用 `elif s[i]==m` 将该字母加入结果。
