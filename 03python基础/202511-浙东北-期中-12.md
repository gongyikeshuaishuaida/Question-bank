# 题目 12

有如下 Python 程序段：

```python
import random
a = [0] * 6
for i in range(len(a)):
    x = random.randint(3, 19)
    if i % 2 == 0:
        t = x
        while t % 2 == 0 and t > 0:
            t = t // 2
        a[i] = t
    else:
        a[i] = a[i-1] + x
print(a)
```

执行该程序段后，变量 `a` 的值可能是（   ）

A. `[7, 17, 8, 14, 11, 19]`  
B. `[9, 11, 5, 25, 7, 15]`  
C. `[21, 25, 9, 22, 13, 30]`  
D. `[5, 18, 9, 14, 11, 30]`

---

## 答案

**正确答案：** D

---

## 解析

偶数下标处的值由 `x` 不断除以 2 得到，必为 3 到 19 范围内的奇数，且不可能为 21，因此 A 中 8、C 中 21 不可能。奇数下标处为前一项加上 3 到 19 的随机数。B 的第二项 `11-9=2` 不在范围内。D 中差值分别为 13、5、19，均可由随机数产生，故可能。
