# 题目 09

有如下 Python 程序段：

```python
a = [1, 1, 2, 3, 5, 8, 4]
b = 0
for i in range(len(a)-1, 1, -2):
    if a[i] == a[i-1] + a[i-2]:
        b = b + 1
print(b)
```

执行该程序段后，输出的结果是（   ）

A. 1  
B. 2  
C. 3  
D. 4

---

## 答案

**正确答案：** B

---

## 解析

`range(6,1,-2)` 中 `i` 依次为 6、4、2。`i=6` 时 `4 != 8+5`；`i=4` 时 `5=3+2`，计数加 1；`i=2` 时 `2=1+1`，计数再加 1。最终输出 2。
