# 题目 10

某 Python 程序如下：

```python
s = "ABxy#Cd"
t = ""
flag = True
for i in range(len(s)):
    if "a" <= s[i] <= "z" and flag:
        x = chr(((ord(s[i])-95))%26+97)
    elif "A" <= s[i] <= "Z" and flag:
        x = chr(((ord(s[i])-66))%26+65)
        flag = False
    else:
        x = s[i]
        flag = True
    t = t + x
print(t)
```

执行该程序后，输出值为（   ）

- A. `ZBza#Bd`
- B. `ZAza#Bf`
- C. `ZBxy#Cd`
- D. `ZBzy#Bd`

---

## 答案

**正确答案：** A

---

## 解析

`flag=True` 时，大写字母按表达式变为前一位，小写字母变为后一位；若 `flag=False` 或遇到非字母则原样拼接并令 `flag=True`。依次处理 `"ABxy#Cd"` 得到 `Z`、`B`、`z`、`a`、`#`、`B`、`d`，即 `ZBza#Bd`。
