# 题目 08

下列代码中不能实现变量 `k` 在数值 1 和 0 之间切换的是（   ）

- A. `k=1-k`
- B. `k=(1+k)%2`
- C. `dic1={1:"0",0:"1"};k=dic1[k]`
- D. `s=[0,1];k=s[(k+1)%2]`

---

## 答案

**正确答案：** C

---

## 解析

A、B、D 都能在整数 `0` 和 `1` 之间切换。C 中字典的值是字符串 `"0"` 和 `"1"`，执行后 `k` 变成字符串而不是数值 0 或 1，不能实现变量在数值 1 和 0 之间切换。
