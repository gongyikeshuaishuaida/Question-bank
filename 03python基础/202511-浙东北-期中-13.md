# 题目 13

某班级期末考试数据已经统计出来，存储在一个 Python 列表中，包含每位学生的学号、姓名和总分。现在需要编写一个程序，根据每个学生的总分计算出其在班级里的名次，并输出班级第一名学生姓名。请回答下列问题：

**（1）** 若学生数据列表为 `lst = [[1, "小明", 523], [2, "小刚", 575], [3, "小红", 566], [4, "小华", 523]]`，程序运行结束后，存放最终排名的列表 `rank` 的值应为 ______________。

**（2）** 实现上述排名功能的部分 Python 程序如下，请在划线处填入合适的代码。

```python
# 将全班学生数据存入lst列表，代码略
____①____
rank = [0] * n
for i in range(n):
    count = 0
    for j in range(n):
        if lst[j][2] > ____②____:
            count = count + 1
    ____③____ = count + 1
print("最终排名列表:", rank)
for i in range(n):
    if rank[i] == 1:
        print("班级第一名是:", ____④____)
```

---

## 答案

**正确答案：** （1）`[3, 1, 2, 3]`；（2）① `n=len(lst)`；② `lst[i][2]`；③ `rank[i]`；④ `lst[i][1]`

---

## 解析

排名按“比当前学生分数高的人数 + 1”计算。575 分为第 1 名，566 分为第 2 名，两个 523 分并列第 3 名，所以 `rank` 为 `[3,1,2,3]`。程序中 `n` 是学生人数，比较对象是当前学生 `lst[i][2]`，排名写入 `rank[i]`，第一名姓名是 `lst[i][1]`。
