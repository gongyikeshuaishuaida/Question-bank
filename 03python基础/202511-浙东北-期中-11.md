# 题目 11

某程序如下：

```python
s = input()
sn = c = 0
for char in s:
    if ('A' <= char <= 'Z') or ('a' <= char <= 'z'):
        sn += 1
    elif '0' <= char <= '9':
        if char == '0':
            c = 0
        else:
            c += ord(char) - ord('0')
print(c, sn)
```

程序运行时，若输出结果为 `5 4`，则可能的输入是（   ）

A. `Py2t0h3n`  
B. `Z0d3b2m`  
C. `M5N0P2Q`  
D. `jmS0Zz22`

---

## 答案

**正确答案：** B

---

## 解析

`sn` 统计字母个数，`c` 累加数字值但遇到 `0` 会清零。B 中有 4 个字母 `Z,d,b,m`，数字处理为先遇到 0 清零，再加 3 和 2，最终 `c=5`，所以输出 `5 4`。
