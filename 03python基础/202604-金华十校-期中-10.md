# 题目 10

有如下程序段：

```python
s = input（   ）
num = 0
s1 = ""
for i in range(len(s)):
    if s[i] == 'e':
        num += 1
    else:
        if num > 0:
            num -= 1
        else:
            s1 += s[i]
```

若输入的 `s` 为 `"excellent"`，执行该程序段后，`s1` 的值为（   ）

- A. `"xcellent"`
- B. `"cllent"`
- C. `"clent"`
- D. `"clt"`

---

## 答案

**正确答案：** D

---

## 解析

字符 `e` 会使 `num` 加 1，之后遇到非 `e` 字符时若 `num>0` 就抵消一次而不加入 `s1`。跟踪 `"excellent"`：第 1 个 `e` 抵消 `x`，第 4 个 `e` 抵消第二个 `l`，第 7 个 `e` 抵消 `n`，剩余加入 `s1` 的字符为 `c`、`l`、`t`，所以结果是 `"clt"`。
