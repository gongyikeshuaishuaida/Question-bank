# 题目 09

有一个室温监测系统，用变量 `n` 表示温度状态。当室温小于 20℃ 时，`n` 的值为 1；当室温大于等于 20℃ 并且小于 30℃ 时，`n` 的值为 2；当室温大于等于 30℃ 时，`n` 的值为 3。变量 `t` 存储当前室温，下列程序段能正确表示上述情况的有（   ）

①

```python
if t < 20:
    n = 1
if t < 30:
    n = 2
else:
    n = 3
```

②

```python
if t < 20:
    n = 1
if 20 <= t < 30:
    n = 2
if t >= 30:
    n = 3
```

③

```python
n = 3
if t < 20:
    n = 1
elif t < 30:
    n = 2
```

④

```python
if t < 30:
    n = 2
elif t < 20:
    n = 1
else:
    n = 3
```

- A. ①②
- B. ①④
- C. ②③
- D. ③④

---

## 答案

**正确答案：** C

---

## 解析

②分别判断三个互斥区间，③先默认 `n=3`，再处理小于 20 和小于 30 的情况，均正确。①中 `t<20` 时第二个 `if t<30` 会把 `n` 改为 2；④中 `t<20` 也先满足 `t<30`，会被误设为 2。
