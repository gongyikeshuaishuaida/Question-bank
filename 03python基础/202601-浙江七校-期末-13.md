# 题目 13

小张设计了一个股票价格模拟分析算法来监测并提醒买入信号。给定股票价格序列，首先识别该序列的所有波峰（局部最大值）和波谷（局部最小值），例如：当 `a[i]` 分别大于 `a[i-1]` 和 `a[i+1]`，则 `a[i]` 为波峰；反之当 `a[i]` 分别小于 `a[i-1]` 和 `a[i+1]`，则 `a[i]` 为波谷。如果出现以下任意一个条件，则触发提醒买入信号：

条件一：出现连续 3 个递增波峰，第 3 个波峰触发买入信号。
条件二：出现大于第一个波峰的波谷，该波谷触发买入信号。

请回答下列问题：

（1）若股票价格序列为 `[95,100,98,92,95,102,104,100,103,98]`，____（填：是/否）会触发买入信号。

（2）实现买入信号检测功能的 Python 程序如下，请在划线处填入合适的代码。

```python
n = len(prices)
peaks, valleys = [], []
for i in range(1, n-1):
    pre = prices[i-1]
    cur = prices[i]
    nex = prices[i+1]
    if ①:
        peaks.append([i, cur])
    elif cur < pre and cur < nex:
        valleys.append([i, cur])
buy_signal = False
pos = 0
if len(peaks) >= 3:
    for i in range(②):
        if peaks[i-2][1] < peaks[i-1][1] < peaks[i][1]:
            buy_signal = True
            pos = i
            break
if not buy_signal and peaks and valleys:
    first_peak_price = peaks[0][1]
    for i in range(len(valleys)):
        if valleys[i][1] > first_peak_price:
            buy_signal = True
            pos = i
            break
# 输出是否触发买入信号，代码略
```

（3）加框处语句有误，请改正。

---

## 答案

（1）否

（2）① `cur > pre and cur > nex`；② `2, len(peaks)`

（3）`first_peak_price = peaks[0][1]`

---

## 解析

（1）该序列的波峰不是连续 3 个递增，波谷也没有大于第一个波峰 100。② 从第 3 个波峰开始检查三连递增，故下标从 2 到 `len(peaks)-1`。加框语句应取第一个波峰价格而不是整个记录。
