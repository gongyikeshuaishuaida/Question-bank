# 题目 10

有如下 Python 程序段：

```python
n=0
for c in s:
    if n==0:
        x=c
    if x==c:
        n+=1
    else:
        n-=1
```

若 s 为 `"甲丙乙丙丙丙丁"`，执行该程序段后，x 的值为（   ）

- A. "甲"
- B. "乙"
- C. "丙"
- D. "丁"

---

## 答案

**正确答案：** C

---

## 解析

程序类似“候选字符计数”过程。依次处理字符串后，`n` 为 0 时会重置 `x`，后续连续多个“丙”使 `x` 稳定为“丙”，最后遇到“丁”只使计数减 1，不改变 `x`，因此最终 `x="丙"`。
