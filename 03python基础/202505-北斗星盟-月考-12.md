# 题目 12

有下列Python程序段，当 `n=3`，`s="abcdefghij"` 时，该程序段执行后，`rows[0]` 的值是（   ）

```python
rows = [""] * n; cur_row = 0; direction = 1
for char in s:
    rows[cur_row] += char
    cur_row += direction
    if cur_row == -1 or cur_row == n:
        cur_row -= direction
        direction = -direction
```

- A. `"abc"`
- B. `"aei"`
- C. `"afg"`
- D. `"adgj"`

---

## 答案

**正确答案：** C

---

## 解析

当行号越界时，程序先把 `cur_row` 回退到边界行，再改变方向，因此边界行会连续接收字符。执行过程得到 `rows[0]="afg"`、`rows[1]="beh"`、`rows[2]="cdij"`，所以 `rows[0]` 为 `"afg"`。
