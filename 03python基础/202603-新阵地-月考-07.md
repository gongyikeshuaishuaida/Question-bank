# 题目 07

有如下 Python 程序段：

```python
res, i, t = "", 0, 1
s = input()
while i < len(s) and t >= 0:
    ch = s[i]
    if ch == "@":
        t -= 1
    elif ch == "$":
        t += 1
    else:
        res += ch
    i += 1
```

若输入 `s` 为 `"a@b$c@d@e"`，执行该程序段后，`res` 的值为（   ）

A. `"abc"`  
B. `"abcd"`  
C. `"ab"`  
D. `"a"`

---

## 答案

**正确答案：** B

---

## 解析

程序按顺序扫描字符串，遇到普通字符就加入 `res`，遇到 `@` 使 `t` 减 1，遇到 `$` 使 `t` 加 1。当处理到第二个 `@` 之前，已经加入的普通字符为 `a`、`b`、`c`、`d`；第二个 `@` 处理后 `t` 变为 `-1`，循环结束，不再处理 `e`，所以 `res` 为 `"abcd"`。
