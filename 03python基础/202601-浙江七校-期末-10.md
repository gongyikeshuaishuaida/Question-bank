# 题目 10

有如下 Python 程序段：

```python
def fun(s1, s2):
    p = 0
    for i in range(len(s1)):
        if s1[i] == s2[p]:
            p += 1
        if p == len(s2):
            return True
    return False
```

`s1` 和 `s2` 分别取下列值时，调用函数得到的结果与其他三项不同的是（    ）
- A. `"12345"`, `"135"`
- B. `"36425"`, `"346"`
- C. `"98765"`, `"876"`
- D. `"24680"`, `"246"`

---

## 答案

**正确答案：** B

---

## 解析

函数判断 `s2` 是否为 `s1` 的子序列。A、C、D 都能按顺序匹配成功，B 中 `346` 不能在 `36425` 中按顺序匹配。
