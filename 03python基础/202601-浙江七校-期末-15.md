# 题目 15

某通信系统对一个明文字符串 `s` 进行加密和解密操作。加密过程包含两个步骤：

步骤一：翻转。将明文 `s` 中多段连续的区间进行翻转。区间存储在密钥 `key1` 中，每个元素为 `[st, ed]`，若 `st<=ed`，表示将 `s` 到 `ed` 的区间（即 `s[st:ed+1]`）进行翻转；若 `st>ed`，表示将 `st` 到字符串末尾再连接字符串开头到 `ed` 的完整区间（即 `s[st:]+s[:ed+1]`）进行翻转。

步骤二：旋转。将步骤一得到结果划分成若干个长度为 `L` 的连续子串，对每个子串分别循环右移 `X` 位（包括最后一个长度可能不足 `L` 的子串），然后重新拼接。密钥 `key2` 的每个元素表示一次旋转操作（即 `[L, X]`），如果 `key2` 中有多个元素，则表示进行了多轮旋转操作。

例如，明文 `s` 为 `"Hello"`，密钥 `key1` 为 `[[1,4],[3,1]]`，密钥 `key2` 为 `[[2,1]]`，加密过程如图所示。

请回答下列问题：

（1）若明文 `s` 为 `"transform"`，密钥 `key1` 为 `[[2,5]]`，密钥 `key2` 为 `[[4,1],[8,2]]`，请问加密后得到的密文第一个字符为____。

（2）有如下 `get_length()` 和 `rev()` 两个函数，其中 `rev()` 函数加框处的语句有误，请改正。

```python
def get_length(n, st, ed):
    return (ed - st + n) % n + 1

def rev(data, st, ed):
    n = len(data)
    t = get_length(n, st, ed)
    for i in range(t):
        p = (st + i) % n
        q = (ed - i) % n
        data[p], data[q] = data[q], data[p]
```

（3）实现加密和解密的代码 Python 程序如下，请在划线处填入合适的代码。

```python

def ring_shift(data, st, ed, sftlen):
```

## 答案

（1）`a`

（2）`t = get_length(n, st, ed) // 2`

（3）① `rev(data, key[i][0], key[i][1])`；② `rev(data, st, ed)`；③ `shift = le - key[i][1]`

---

## 解析

区间翻转时只需交换区间长度的一半，否则会再次交换回去。循环右移可用三次翻转实现；解密旋转等价于右移 `le - key[i][1]` 位。
