# 题目 13

现有一辆卡车需要装货，各个货物的重量数据存储在数组 `a` 中，卡车载重为 `weight`，搬运时必须选择连续排列的货物，要求解不超过卡车载重量且最大载重的货物组合，若有多个组合满足要求则选取最后一个。

**（1）** 若货物重量为 `[20,10,50,70,25,35,30,5,80]`，卡车载重为 `100`，则选取的货物重量分别是 ______

**（2）** 实现上述功能的 Python 程序如下，请在划线处填入合适代码。

```python
a = [20,10,50,70,25,35,30,5,80]
weight = 100
i = j = s = 0
besti = bestj = bests = 0
while ____①____:
    s += a[j]
    j += 1
    while i <= j and s > weight:
        ____②____
        i += 1
    if ____③____:
        bests = s
        besti = i
        bestj = j
print(bests,besti,bestj)
```

---

## 答案

**正确答案：** （1）25、35、30、5；（2）① `j<len(a)`；② `s-=a[i]`；③ `s>=bests`

---

## 解析

程序维护区间 `[i,j)` 的连续货物总重 `s`。当 `s` 超过载重时，不断从左端移出货物，即执行 `s-=a[i]` 并右移 `i`。若当前总重不超过载重且 `s>=bests`，就更新最优解；使用 `>=` 可在总重相同时保留后出现的组合。样例中总重为 95 的最后最优连续组合是 `25,35,30,5`。
