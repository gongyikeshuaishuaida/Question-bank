# 题目 13

有若干辆相同的货车，其最大载重为 `lim`。现有一批重量不一的货物按到货时间依次放置在一条传送带上，货物的重量按到货时间先后依次存储在列表 `a` 中，货车装载时只能按到货时间依次取货（先到先取），现要求出装载这批货物最少需要的货车数量及各车装载情况（单件货物的重量不超过货车最大载重），请回答下列问题。

**（1）** 若 `a=[3,1,6,8,5,4]`，货车最大载重 `lim=10`，则最少需要 ______ 辆货车。

**（2）** 实现上述功能的 Python 代码如下，请在划线处填入合适的代码。

```python
# 获取货物重量，按到货时间先后依次记录在列表a中，代码略
lim = int(input("请输入货车最大载重: "))
n = ____①____
st, cnt, tot = 0, 0, 0
for i in range(n):
    tot += a[i]
    if tot > lim:
        cnt += 1
        print("第", cnt, "辆车装载的货物编号为: ", st, "->", i - 1)
        tot = ____②____
        st = i
____③____
print("第", cnt, "辆车装载的货物编号为: ", st, "->", n - 1)
```

---

## 答案

**正确答案：** （1）3；（2）① `len(a)`；② `a[i]`；③ `cnt+=1`

---

## 解析

按到货顺序装车，当前车累计重量超过 `lim` 时，前一辆车结束，当前货物应作为新车的第一件货物，因此 `tot=a[i]`，起点更新为 `i`。循环结束后还要统计最后一辆车，所以执行 `cnt+=1`。样例分为 `[3,1,6]`、`[8]`、`[5,4]`，共 3 辆。
