# 题目 10

有如下 Python 程序段：

```python
s = "banana"
c = ""
i = 0
t = 2
while i < len(s) and t > 0:
    if s[i] == "a":
        t -= 1
    else:
        c = s[i] + c
    i += 1
print(c)
```

执行该程序段后，输出的结果是（   ）

- A. `"nb"`
- B. `"bn"`
- C. `"nan"`
- D. `"nnb"`

---

## 答案

**正确答案：** A

---

## 解析

循环在遇到第 2 个 `"a"` 后结束。处理 `"banana"` 的前 4 个字符：`b` 不是 `"a"`，`c="b"`；第 1 个 `a` 使 `t=1`；`n` 不是 `"a"`，执行 `c="n"+"b"` 得到 `"nb"`；第 2 个 `a` 使 `t=0`，循环结束，输出 `"nb"`。
