# 题目 09

某二叉树有 `a`、`b`、`c`、`d`、`e` 五个节点，若前序遍历序列为 `abcde`，后序遍历序列为 `bdeca`，则该二叉树的树形结构可能为（   ）

![[attachments/202604金华十校_09_图1.png]]

---

## 答案

**正确答案：** C

---

## 解析

前序遍历第一个节点是根节点 `a`，后序遍历最后一个节点也是根节点 `a`。前序中 `a` 后为 `b`，后序中 `b` 最先出现，说明 `b` 可作为一个叶子子树；剩余节点形成以 `c` 为根、`d` 和 `e` 为子节点的子树。选项 C 的前序为 `a b c d e`，后序为 `b d e c a`，与题设一致。
