# 题目 12

有如下 Python 程序段：

```python
stks, cnt = [], [0] * 5
for i in range(len(a)):
    if i % 3 != 2:
        if cnt[a[i]] == len(stks):
            stks.append([a[i]])
        else:
            stks[cnt[a[i]]].append(a[i])
        cnt[a[i]] += 1
    else:
        val = stks[-1].pop()
        if not stks[-1]:
            stks.pop()
        cnt[val] -= 1
```

若列表 `a` 为 `[2,2,0,3,1,4,4,1,3]`，程序运行结束时 `stks[0]` 不包含（    ）
- A. 1
- B. 2
- C. 3
- D. 4

---

## 答案

**正确答案：** A

---

## 解析

按程序逐项模拟入栈和每第三项出栈，最终底层列表中不包含元素 1。
