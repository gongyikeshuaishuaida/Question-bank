# 题库索引

本索引通过 Dataview 动态读取所有含 `id`、`试卷`、`知识点` 字段的题目笔记。

## 按知识点查询

### 01数据与编码

```dataview
TABLE file.link AS 题目, 试卷, 题号, 题型, 难度, 状态
WHERE id AND 试卷 AND 知识点 AND contains(知识点, "01数据与编码")
SORT 试卷 ASC, 题号 ASC
```

## 按目录统计

```dataview
TABLE length(rows) AS 数量
WHERE id AND 试卷
GROUP BY file.folder
SORT file.folder ASC
```
