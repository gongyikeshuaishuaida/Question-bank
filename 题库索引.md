# 题库索引

本索引通过 Dataview 动态读取所有含 `id`、`试卷`、`知识点` 字段的题目笔记。

## 按知识点查询

```dataview
TABLE rows.file.link AS 题目
WHERE id AND 试卷 AND 知识点
FLATTEN 知识点 AS 知识点
GROUP BY 知识点
SORT key ASC
```

## 按目录统计

```dataview
TABLE length(rows) AS 数量
WHERE id AND 试卷
GROUP BY file.folder
SORT file.folder ASC
```

## 图片待核验

```dataview
TABLE file.link AS 题目, 试卷, 题号, 知识点
WHERE id AND 图片核验 = "待核验"
SORT 试卷 ASC, 题号 ASC
```
