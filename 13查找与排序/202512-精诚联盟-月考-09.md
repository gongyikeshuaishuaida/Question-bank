# 题目 09

某对分查找法的Python 程序段如下：

```python
i = 0; j = 9
while i <= j:
    m = (i+j+1)//2   #①
    if a[m] > key:
        j = m-1
    else:
        i = m+1
```

数组a[0]到a[9]中存放着各不相同且按升序排列的元素，若查找键 key 在以下范围时，则①处语句“m = (i+j+1)//2”被执行次数不同于其他项的是

A. key<a[0]
B. a[2]<=key<=a[3]
C. key=a[5]
D. a[7]<=key<=a[8]

---

## 答案

**正确答案：** D

---

## 解析

该程序每次取偏右中点，按 `a[m] > key` 缩小右边界，否则缩小左边界。对 A、B、C 三种情况跟踪边界变化，①处均执行 4 次；当 `a[7]<=key<=a[8]` 时，边界变化更快，①处执行 3 次，因此 D 不同。
