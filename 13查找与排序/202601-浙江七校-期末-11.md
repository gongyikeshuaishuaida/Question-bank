# 题目 11

有如下 Python 程序：

```python
def b_search(a, key, left, right):
    while left <= right:
        mid = (left + right) // 2
        if a[mid] > key:
            right = mid - 1
        else:
            left = mid + 1
    return ①

for i in range(1, len(a)):
    key = a[i]
    pos = b_search(a, key, 0, i-1)
    for j in range(②):
        a[j+1] = a[j]
    a[pos] = key
```

为实现数组 `a` 升序排序，则程序划线①②处应填入的语句为（    ）
- A. ① `left` ② `i-1, pos, -1`
- B. ① `right` ② `i-1, pos, -1`
- C. ① `left` ② `i-1, pos-1, -1`
- D. ① `right` ② `i-1, pos-1, -1`

---

## 答案

**正确答案：** C

---

## 解析

二分查找结束时 `left` 是插入位置；插入排序右移元素时需从 `i-1` 到 `pos`，因此 `range(i-1, pos-1, -1)`。
