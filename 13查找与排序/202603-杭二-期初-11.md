# 题目 11

有如下 Python 程序段：

```python
a = [18,30,21,28,25,23]
target = 24
for i in range(len(a) - 1):
    k = i
    for j in range(i + 1, len(a)):
        if abs(a[j] - target) < abs(a[k] - target):
            k = j
    if k != i:
        a[i], a[k] = a[k], a[i]
```

执行该程序段后，列表 `a` 为（   ）

- A. `[25,23,21,28,18,30]`
- B. `[25,23,30,28,18,21]`
- C. `[25,21,30,28,18,23]`
- D. `[23,25,21,28,18,30]`

---

## 答案

**正确答案：** A

---

## 解析

程序按元素与 `target=24` 的距离从小到大选择排序。25 和 23 距离均为 1，但先找到 25，严格小于才更新，因此前两项为 25、23；后续依次为 21、28、18、30。
