# 题目 11

有如下 Python 程序段：

```python
import random
a = [0] * 5
for i in range(5):
    a[i] = random.randint(1, 5)  # 随机生成1~5的整数
for i in range(4):
    if a[i] < a[i+1]:
        a[i], a[i+1] = a[i+1], a[i]
print(a)
```

执行该程序段后，输出的结果不可能是（   ）

- A. `[5,4,3,1,2]`
- B. `[3,5,4,2,1]`
- C. `[4,5,3,2,1]`
- D. `[5,4,3,2,1]`

---

## 答案

**正确答案：** A

---

## 解析

程序从左到右依次比较相邻元素，若左边小于右边就交换。最后一次比较发生在下标 3 和 4，因此程序结束时一定满足 `a[3] >= a[4]`。选项 A 的末两项为 `1,2`，不满足这一条件，所以不可能输出。
