# 题目 12

有如下 Python 程序段：

```python
from random import randint
from math import sqrt

a = []; n = 6
for i in range(n):
    a.append(randint(1, 10))
for i in range(n - 1):
    for j in range(n - 1, i, -1):
        if a[j] > a[j - 1] and sqrt(a[j]) != int(sqrt(a[j])):
            a[j], a[j - 1] = a[j - 1], a[j]
print(a)
```

执行该程序段后，输出结果不可能为（    ）
- A. `[5, 9, 7, 2, 4, 1]`
- B. `[5, 4, 3, 1, 9, 3]`
- C. `[10, 8, 3, 2, 4, 9]`
- D. `[9, 7, 4, 8, 5, 2]`

---

## 答案

**正确答案：** D

---

## 解析

非完全平方数在满足较大条件时会向前交换，完全平方数不主动前移；选项 D 的相对位置不可能由该过程得到。
