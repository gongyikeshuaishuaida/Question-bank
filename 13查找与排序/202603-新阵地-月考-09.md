# 题目 09

编写程序，实现将 `tmp` 值插入到升序列表 `a` 中，并保持升序，程序段如下：（下表为函数功能）

```python
if tmp >= a[-1]:
    a.append(tmp)
else:
    j = len(a) - 1
    while j >= 0 and tmp < a[j]:
        if ____(1)____:
            a.append(a[j])
        else:
            ____(2)____
        j -= 1
    ____(3)____
```

划线（1）（2）（3）处可供选择的语句有：

① `j==len(a)-1`  
② `j==0`  
③ `a[j+1]=a[j]`  
④ `a[j]=a[j-1]`  
⑤ `a[j]=tmp`  
⑥ `a[j+1]=tmp`

下列选项中，填入划线处能使程序正确运行的顺序是（   ）

A. ①③⑥  
B. ①④⑤  
C. ②④⑤  
D. ②③⑥

---

## 答案

**正确答案：** A

---

## 解析

从列表尾部向前移动元素。第一次移动尾元素时，需要用 `a.append(a[j])` 扩展列表，所以（1）应为 `j==len(a)-1`；之后把当前元素后移一位，故（2）为 `a[j+1]=a[j]`；循环结束时，`tmp` 应放入空出的 `j+1` 位置，故（3）为 `a[j+1]=tmp`。顺序为①③⑥。
