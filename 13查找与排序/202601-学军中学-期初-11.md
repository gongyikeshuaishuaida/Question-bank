# 题目 11

某二分查找算法的 Python 程序段如下：

```python
import random
key = random.randint(0, 4) * 2 + 5
ans = 0
a = [3, 5, 5, 8, 9, 11, 11, 13, 15, 17]
i, j = 0, len(a) - 1
while i <= j:
    m = (i + j + 1) // 2
    if a[m] <= key:
        i = m + 1
    else:
        j = m - 1
    ans += a[m]
```

程序运行后，`ans` 的值不可能的是（    ）
- A. 24
- B. 25
- C. 33
- D. 39

---

## 答案

**正确答案：** A

---

## 解析

`key` 只可能为 5、7、9、11、13，按程序二分路径累加 `a[m]`，枚举可得不会出现 24。
