# 题目 11

为实现对数组数据进行升序排序，编写如下 python 程序段：

```python
a = [12,55,37,45,2,80,94,21]
for i in range(len(a)-1):
    for j in range(______________):
        if a[j] > a[j+1]:
            a[j], a[j+1] = a[j+1], a[j]
print(a)
```

划线处应填入的代码不正确的是（   ）

- A. `len(a)-i-1`
- B. `i,-1,-1`
- C. `len(a)-2,i-1,-1`
- D. `i,len(a)-1-i`

---

## 答案

**正确答案：** D

---

## 解析

冒泡排序每轮应覆盖待排序部分的相邻比较。A 是常见从前往后的冒泡范围；B、C 可从后往前逐步比较。D 的起点和终点都随 `i` 向中间收缩，会漏掉必要比较，不能保证完整升序排序。
