# 题目 11

执行下列Python程序段，若 `a` 的初值为 `[4, 6, 10, 6, 7, 8, 2, 5, 8, 10]`，则输出结果是（   ）

```python
count = 1; max_len = 1; st = 0
for i in range(1,len(a)):
    if a[i] > a[i-1]:
        count += 1
    else:
        if max_len <= count:
            max_len = count
            st = i-count
        count = 1
print(a[st:st+max_len])
```

- A. `[4, 6, 10]`
- B. `[6, 7, 8]`
- C. `[2, 5, 8]`
- D. `[2, 5, 8, 10]`

---

## 答案

**正确答案：** B

---

## 解析

程序只在遇到非递增位置时更新 `max_len` 和 `st`，末尾的递增段不会再被单独比较。第一个递增段 `[4,6,10]` 长度为 3；遇到 `2` 时，上一段 `[6,7,8]` 长度也为 3，因条件是 `max_len <= count`，会把起点更新为该段起点，所以输出 `[6, 7, 8]`。
