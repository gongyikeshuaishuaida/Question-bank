# 题目 08

有如下 Python 程序段：

```python
a = [5, 8, 6, 4, 2, 5, 4, 5, 8, 3, 1, 6, 3]
b = [0, 2, 3, 6, 7, 8, 9, 12]
k = 0; m = 0
for i in range(b[len(b)-2]):
    if i == b[k]:
        k += 2
    j = b[k]
    while j <= b[k+1]:
        if a[i]+a[j] < m or m == 0:
            m = a[i]+a[j]
        j += 1
```

运行程序段后，`m` 的值是（   ）

A. 4  
B. 6  
C. 7  
D. 8

---

## 答案

**正确答案：** B

---

## 解析

循环中 `i` 的范围是 `0` 到 `8`。`k` 会在 `i` 到达 `b[k]` 时跳到下一组区间，`j` 在当前区间内枚举。最小和先在 `i=0,j=4` 时更新为 `5+2=7`，后来在 `i=7,j=10` 时更新为 `5+1=6`，之后没有更小值，因此最终 `m=6`。
