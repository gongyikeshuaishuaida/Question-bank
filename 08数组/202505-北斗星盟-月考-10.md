# 题目 10

数组元素 `d[0]` 到 `d[n-1]` 已按升序排列。现需要通过删除操作处理数组，使得重复数据只保留一个，并保持数组的有序性不变。实现该功能的程序段如下，方框中应填入的正确代码为（   ）

```python
i = 0; j = 1
while j < len(d):
    if d[j] != d[i]:
        ______
        i += 1
    j += 1
print(d[:i+1])
```

- A. `d[i-1] = d[j]`
- B. `d[j-1] = d[j]`
- C. `d[i+1] = d[j]`
- D. `d[i] = d[j]`

---

## 答案

**正确答案：** C

---

## 解析

变量 `i` 指向当前已保留的不重复数据的最后位置，`j` 扫描后续元素。当 `d[j] != d[i]` 时，应先把新的不重复元素放到 `i+1` 位置，再令 `i += 1`，所以应填 `d[i+1] = d[j]`。
