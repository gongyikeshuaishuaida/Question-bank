# 题目 12

某 Python 程序如下：

```python
a = [9,7,5,8,4,6,3,2]
n = 8
num = [0] * n
i = 0
while i < n:
    num[i] = 1
    for j in range(0,i):
        if a[j] > a[i] and num[j] >= num[i]:
            num[i] = num[j] + 1
    i = i + 1
ans = 0
for i in range(n):
    if num[i] > ans:
        ans = num[i]
print(ans)
```

执行输出结果为（   ）

- A. 5
- B. 4
- C. 6
- D. 3

---

## 答案

**正确答案：** C

---

## 解析

`num[i]` 表示以 `a[i]` 结尾、满足前面元素大于后面元素的最长序列长度。对数组 `[9,7,5,8,4,6,3,2]` 跟踪可得最大长度为 6，例如 `9,7,5,4,3,2`，因此输出 6。
